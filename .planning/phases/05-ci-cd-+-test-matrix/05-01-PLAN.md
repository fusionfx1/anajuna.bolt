# Phase 05 Plan 01: CI Setup

## Objective

Stand up the full CI gate: GitHub Actions workflow (typecheck + lint + unit + build + E2E),
a CI-safe E2E smoke test, a static RLS contract test, and updated .gitignore.

## Tasks

### Task 1 — GitHub Actions CI workflow (CI-01, CI-05)
type: auto

Create `.github/workflows/ci.yml` with jobs:
- `typecheck`: `npm run typecheck`
- `lint`: `npm run lint`
- `test`: `npm test -- --reporter=verbose`
- `build`: `npm run build` with placeholder Supabase env vars; posts bundle size to step summary
- `e2e`: downloads build artifact, installs Playwright browsers, runs `npx playwright test`

All jobs run on `ubuntu-latest`, Node 20, with npm cache. E2E depends on build artifact.

### Task 2 — Playwright E2E smoke test + config update (CI-02)
type: auto

Update `playwright.config.ts`:
- Change baseURL to `http://localhost:4173` (preview server port)
- Add `webServer` block: `npm run preview`, url 4173, reuseExistingServer in dev
- Set `forbidOnly: !!process.env.CI`, `retries: CI ? 2 : 0`, `workers: CI ? 1 : undefined`

Create `e2e/agent-feed.spec.ts` with 3 CI-safe structural smoke tests:
1. Page loads without crashing (no fatal JS errors)
2. Body has visible content (not blank)
3. Login form visible when real Supabase URL provided (skip in CI with placeholder)

### Task 3 — RLS policy static contract test (CI-03)
type: auto

Create `src/hooks/__tests__/rls-policy-contract.test.ts`:
- Reads `supabase/migrations/20260501_agent_decisions_rls_contract.sql`
- Asserts file exists, contains expected USING clause, policy name, DROP statement, TO authenticated
- Documents v1 gap: static check only, not a live policy enforcement test

### Task 4 — .gitignore CI artifact exclusions (CI-04)
type: auto

Append to root `.gitignore`:
```
# CI / test artifacts
playwright-report/
test-results/
.playwright/
```

`graphify-out/cache/` already present. Python __pycache__ covered by trading_system/.gitignore.
