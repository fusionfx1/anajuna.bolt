| Token | px | Tailwind class | Usage |
|-------|----|----------------|-------|

| sm | 8px | gap-2, p-2 | Header icon gap, dense compound fields |
| md | 12–16px | gap-3/gap-4, py-3, px-3 | Field row, button groups, input padding |
| lg | 20px | p-5 | Section card padding |
| xl | 20px | mb-5 | Section header bottom spacing |

**Exceptions:**
- Touch targets for toggle buttons: minimum `w-11 h-6` (44px × 24px) to meet WCAG 2.5.5

## SecretInput Component Spec

**Source decision:** D-17 (extract from `DataFeedConfig.tsx` lines 87–110), D-18 (Alpaca Key ID must be masked).

**Target location:** `src/components/ui/SecretInput.tsx`

**Interface:**

```tsx
interface SecretInputProps {
  id: string;         // required — ties to htmlFor on label
  label: string;      // displayed uppercase, 12px, slate-400
  value: string;
  onChange: (v: string) => void;
  placeholder?: string;
  hint?: string;      // optional hint below input (12px slate-600)
  disabled?: boolean;
}
```

**Rendered structure:**

```tsx
<div>
  <label
    htmlFor={id}
    className="block text-xs font-semibold text-slate-400 mb-2 uppercase tracking-wide"
  >
    {label}
  </label>
  <div className="relative">
    <input
      id={id}
      type={show ? 'text' : 'password'}
      value={value}
      onChange={e => onChange(e.target.value)}
      placeholder={placeholder}
      disabled={disabled}
      className="w-full bg-slate-800 border border-slate-700 rounded-lg px-3 py-3 text-sm text-white placeholder-slate-500 font-mono
                 focus:outline-none focus:border-emerald-500/50
                 disabled:opacity-50 disabled:cursor-not-allowed
                 pr-10"
    />
    <button
      type="button"
      onClick={() => setShow(v => !v)}
      aria-label={show ? `Hide ${label}` : `Show ${label}`}
      className="absolute right-3 top-1/2 -translate-y-1/2 text-slate-500 hover:text-slate-300 focus:outline-none focus:text-slate-300"
    >
      {show ? <EyeOff size={14} /> : <Eye size={14} />}
    </button>
  </div>
  {hint && (
    <p className="text-xs text-slate-600 mt-1">{hint}</p>
  )}
</div>
```

**Consumers after extraction:**
- `DataFeedConfig.tsx`: Polygon API Key, Alpaca Data Key, Alpaca Data Secret, Alpaca Secret Key (Broker), OANDA API Token
- `DataProvidersSettings.tsx` (refactored): EODHD API Key, Tiingo API Key
- `Settings.tsx` (MT5 password field is removed, so no new consumer here)

**Alpaca Key ID change (D-18):** The `alpacaKeyId` field currently uses `type="text"`. It must be changed to `<SecretInput id="alpaca-key-id" label="Alpaca API Key ID (Broker)" .../>`.

| Python Agents section subtitle | "Configure in your .env file — these keys never reach the browser" |

**Destructive confirmation — Clear Cache:**

Replace `window.confirm()` with an inline confirmation pattern:

1. User clicks "Clear Cache" → button changes to "Confirm Clear?" with red styling + a "Cancel" ghost link beside it.
2. User clicks "Confirm Clear?" → action executes, button shows "Clearing…" → "Cache cleared" for 2s.
3. User clicks "Cancel" → button returns to normal.
